# COMBI MVP

Local-first AI builder scaffold for:

- websites
- web apps
- simple 2D browser games

Current stage: architecture-only skeleton.

## Workspace layout

- `apps/builder-shell` - future UI shell
- `apps/local-engine` - local orchestration server
- `packages/shared` - shared contracts and domain types
- `docs/architecture.md` - module boundaries and pipeline description

## Planned flow

`idea -> type detection -> spec -> file plan -> generation -> preview`

This repository currently contains only the base modular structure and TypeScript scaffolding for the next implementation stages.
